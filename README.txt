# Centro de Comando da Família Almeida Sarcinelli

Esta é a primeira versão do painel familiar, feita de propósito sem frameworks modernos para aumentar a chance de funcionamento no Safari antigo do iPad 2.

## O que já funciona
- Tarefas do dia com escolha de quem concluiu.
- Tarefas da semana com escolha de responsável.
- Lista de compras.
- Revezamento diário das cachorras.
- Salvamento no próprio aparelho usando localStorage.
- Reinício automático das tarefas diárias a cada novo dia.
- Reinício automático das escolhas semanais a cada nova semana.

## Limitação desta primeira versão
Os dados ficam salvos somente no navegador do aparelho em que foram marcados. Eles ainda não sincronizam entre iPad e celulares.

## Como testar no computador
1. Descompacte a pasta.
2. Dê dois cliques em `index.html`.
3. O painel abrirá no navegador.

## Como publicar no GitHub Pages
1. Crie uma conta em https://github.com/
2. Crie um repositório público chamado `centro-comando-familia`.
3. Envie `index.html`, `style.css` e `app.js`.
4. Abra **Settings > Pages**.
5. Em **Build and deployment**, escolha **Deploy from a branch**.
6. Selecione a branch `main` e a pasta `/ (root)`.
7. Salve.
8. Aguarde alguns minutos. O endereço será parecido com:
   `https://SEU-USUARIO.github.io/centro-comando-familia/`

## Como colocar no iPad
1. Abra o endereço publicado no Safari.
2. Toque no botão de compartilhar.
3. Escolha **Adicionar à Tela de Início**.
4. Use o novo ícone como se fosse um aplicativo.

## Segurança
Não coloque dados bancários, senhas, documentos ou informações sensíveis no painel.
